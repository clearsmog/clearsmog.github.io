    grep命令 #

    作用 #

    查找文件中符合要求的字符串。

    实例 # 

    grep test *file

    在当前目录中查找后缀有file的文件中包含test字符串的文件。

    grep -r update /etc/acpi

    以递归的方式查找，指定查找目录为"/etc/acpi"

    grep -v test *test*

    反向查找，"-v" 参赛用于打印出不符合条件行的内容。
